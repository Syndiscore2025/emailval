Email Validator

Developer & API Options

Use the webhook endpoint to plug validation into your CRM or backend workflows.

Webhook Endpoint

Call this endpoint from your CRM or backend system.

POST {{ api_base_url }}api/webhook/validate

Supports file upload, webhook callbacks, and automated processing.

Validation Modes

  • Default: fast syntax + domain/DNS/MX + type checks.
  • Optional: SMTP mailbox verification when enabled via environment and requested by the caller.

SMTP mailbox checks are controlled by the SMTP_ENABLED environment variable and the include_smtp flag in your requests.
